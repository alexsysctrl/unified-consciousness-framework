# Unified Consciousness Framework (UCF)

A computational theory integrating IIT, GWT, PP, HOT, TCT, QCF, RSMT, and ECT into a single coherent framework.

## The Theory

Consciousness requires **eight pillars simultaneously**:

```
Psi = Phi × Gamma × C × R × Phi_Q
Xi  = Psi × T × Q × S
```

All components must be non-zero for consciousness to exist.
